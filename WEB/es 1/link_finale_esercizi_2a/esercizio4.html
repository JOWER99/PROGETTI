Soluzione Esercizio 4: Documento HTML completo

Obiettivo: Creare un documento HTML completo con intestazioni, paragrafi, immagini e link.


<!DOCTYPE html>
<html lang="it">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Il mio primo documento HTML</title>
  </head>
  <body>
    <h1>Benvenuto al mio sito!</h1>
    <p>Questo è un paragrafo di esempio.</p>
    <img src="esempio.jpg" alt="Una bella immagine" width="300">
    <a href="https://www.example.com">Visita il mio sito</a>
  </body>
</html>
